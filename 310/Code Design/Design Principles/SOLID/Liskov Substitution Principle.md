Objects that inherit some super class should be able to replaced by any subclass without breakage

said differently, subclasses can only expand functionality given by the superclass. All functionality of a subclass must be a superset of the super class

Preconditions: Must be unaffected or widened
Postconditions: must be unaffected or narrowed
